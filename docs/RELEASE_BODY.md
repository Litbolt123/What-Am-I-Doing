# What Am I Doing 1.1.5 — Windows installer

## What's new

- **Version bump (updater check):** Use this build to confirm **Settings → Updates (GitHub) → Check for updates…** (or the tray tip after start) correctly sees **1.1.5** as newer than **1.1.4** on a machine where **1.1.4** is already installed. No product changes are intended beyond the version string and installer metadata.

## Notes

- Install **1.1.4** first from its release asset if you are deliberately re-testing the GitHub flow; then publish **1.1.5** and run **Check for updates…** or wait for the automatic check.
